This is the homepage design for a website I created during my internship at a company in Shenzhen, China.
The website includes product browsing, search functionality, and an integrated purchase system. After an online order, request information is automatically sent to the company via email.

The design focuses on clear user guidance, smooth navigation, and a modern tech-oriented visual style to support both product presentation and purchasing flow.
